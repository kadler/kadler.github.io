Terms

  • EBCDIC - Extended Binary Codeded Decimal Interchange Code
  • CCSID - Coded Character Set Identifier
  • CS - Character Set
  • CP - Code Page
  • SBCS - Single Byte Character Set
  • DBCS - Double Byte Character Set
  • MBCS - Mixed Byte Character Set (SBCS + DBCS via SI/SO)
  • SO - Shift Out. In MBCS EBCDIC, switches encoding from SBCS to DBCS
  • SI - Shift In. In MBCS EBCDIC, switches encoding from DBCS to SBCS
  • CECP - Country Extended Code Page
  • ECECP - Euro Country Extended Code Page
